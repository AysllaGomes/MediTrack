### Prerequisites

- [Node.js](https://nodejs.org/en/blog/release/v20.9.0) v.20.9.0
- [NestJS](https://nestjs.com/)

## Installation

1. Clone the repository:

    ```bash
        https://github.com/AysllaGomes/MediTrack.git
    ```
2. Install the dependencies:

    ```bash
      npm install
    ```
3. Running the Application:

    ```bash
    # development
    $ npm run start
    
    # watch mode
    $ npm run start:dev
    
    # production mode
    $ npm run start:prod
    ```

The API will be available at http://localhost:3000.
